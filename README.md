# Android: Netrunner (2012) — Mechanics-First Guide

A mobile-first, accessible website for learning the original 2012 edition of Android: Netrunner.

## Features

- Responsive lesson and section navigation
- Full-guide search
- Reading-progress indicator
- High-contrast dark and light themes
- Print-friendly layout
- Dependency-free static hosting
- Original guide retained verbatim in `guide.md`

## Local preview

Open `index.html` directly, or run a simple static server:

```sh
python -m http.server 8000
```

Then open `http://localhost:8000`.
